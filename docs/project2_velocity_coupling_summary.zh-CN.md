# 项目 2：动态细胞耦合建模原型

## 项目定位

这个项目聚焦于**细胞状态转变过程的计算建模**。

我使用了**肝病进展过程中按疾病阶段分层的单细胞转录组数据**，尝试从横断面观测中提取更接近状态变化结构的信息。具体实现上，我先用固定功能模块表征不同细胞群状态，再在每个细胞类型内部利用伪时间邻域差分近似模块变化速度，随后比较一个细胞群的模块状态能否预测另一个细胞群模块的变化速度，并通过 donor-held-out 双向检验筛选更稳定的方向边。在此基础上，我进一步将候选方向边组织成跨阶段的候选机制链。

这个原型目前还比较初步，但它让我开始更具体地理解：如何定义细胞状态表示，如何从横断面数据中构造近似动态量，以及如何把静态关联推进到更有方向感的结构比较。这些尝试也让我进一步确认，自己希望继续沿着细胞状态建模、方向关系分析和可解释预测这条线深入下去。

## 关键结果

在严格的单向标准下，我筛选出一条更稳定的候选链：

`炎症性内皮 -> 干扰素应答型髓系 -> 炎症型成纤维细胞 -> 肝细胞恶性样程序`

这一结果提示，终末状态变化更可能是在前序微环境重塑过程中被逐步推动出来的，而不是孤立发生的。

## 与刘晓东老师实验室方向的衔接

刘晓东老师实验室关注的核心问题，是细胞命运调控、状态转变机制、多组学整合以及基于人工智能的 Virtual Cell 模型。我的兴趣和现有工作与这一方向具有较明确的契合性：项目 1 使我开始接触基于序列和调控信息的预测建模，项目 2 使我开始尝试对单细胞数据中的状态变化和方向关系进行建模，而我对多组学数据特点的理解，也使我更关注预测结果的生物学解释性。虽然我目前的项目还不是直接建立在干细胞或发育体系上，但所关心的问题与实验室未来在细胞命运预测和状态转变建模上的研究方向是相通的。
