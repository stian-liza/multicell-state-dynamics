# Roadmap

## MVP

- one disease
- one main transition
- one module representation method
- one sparse linear dynamics model
- one coupling mechanism

Success criterion:

- stable module decomposition across seeds
- nontrivial velocity fit on held-out synthetic or real cells
- interpretable high-magnitude module edges

## Phase 2

- multiple cell populations
- explicit spatial neighbor graph
- cross-population coupling matrices
- external reproducibility checks

## Phase 3

- literature-backed priors
- mechanism scoring
- candidate target ranking
- optional perturbation-response simulation
